# Note Generation Prompts

Each file here is a Topic Block. To generate a note:

1. Copy the Base Prompt from `CLAUDE.md` (Note Generation section)
2. Paste `---`
3. Paste the contents of the relevant topic file below
4. Send to Claude

- `druid.md` — Apache Druid  
- `kafka.md` — Apache Kafka

- `react.md` — React.js
- `javascript.md` — JavaScript Deep Dive

## Output
Each prompt generates a `.jsx` file → save in `src/pages/data-engineer/` or relevant folder.
